Welcome to the Account Ledger App!
The purpose of this app is to keep track and submit payments and deposits!

This app is built to update any inventory that you've purchased or deposits put into your account.
There is also a ledger that allows the user to view payments and deposit with certain search parameters.



To use this app you would need to do the following:


Copy this account-ledger-app url.

Open intellij.

Create a new project from version control.

Put the url into the URL bar.



To use this app you need to:


Open up the account-ledger-app project.

Hit run on the console in LedgerAccountApp in intellij.



The file structure for this project is:


LedgerAccountApp -> Main way for the program to run.

DepositsPaymentWriter -> Writes the file for the payments and deposits.

Ledger -> Basic displays for payments and deposits.

CustomSearch -> Allows the user for more precise parameters for searching payments and deposits.

Decor -> Adds some static decorations and a thread.sleeep.

Transaction.csv -> Contains all the transactions.
